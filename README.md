# cs50test

A small HTML/CSS practice repo with standalone pages for experimenting with layout, animation, gradients, cards, and simple static web design.

## Contents

- `test.html` - A styled landing page with a gradient background and button.
- `test2.html` - A larger multi-section page with navigation, cards, and visual effects.

## Run

Open either HTML file directly in a browser:

```bash
start test.html
start test2.html
```

## Notes

No build step or backend is required.
